# 리액트 Ref Of DOM

# Overview

  • Ref를 이용하명 DOM요소에 직접 접근할 수 있다.
  • 특정 태그에 포커싱하거나 form 요소에 입력된 값을 직접 참조할때 사용할 수 있다.
  • 문법을 알고 있으면 이해하는데 전혀 어려움이 없는 내용으로 간단하게 문법만 정리하였다.
  • 공식문서링크

# Syntax

  • createRef API
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
    this.myRef = React.createRef();
  }
  render() {
    return <div ref={this.myRef} />;
  }
}

//Accessing Refs
const node = this.myRef.current;
  • callback pattern
class CustomTextInput extends React.Component {
  constructor(props) {
    super(props);

    this.textInput = null;

    this.setTextInputRef = element => {
      this.textInput = element;
    };
  }

  render() {
    return (
      <div>
        <input
          type="text"
          ref={this.setTextInputRef}
        />
      </div>
    );
  }
}
  • Legacy API (String Refs)
class MyComponent extends React.Component {
  render() {
    return <div ref="myRef" />;
  }
}

//Accessing Refs
const node = this.refs.myRef;
